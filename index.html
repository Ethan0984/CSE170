Why is this week's topic important for teamwork?
This week’s topics were GitHub and the MIM, both of which are important in the
programming field; GitHub is important because you can upload code to a
repository for others to access, so it is an essential tool to know how to use if
you want others to see and edit your code, and vice versa. The MIM allows you to
see how influential you are on yourself, groups, and your community as a whole;
this is important to know so that you can see where your strong areas are, and
also what you may need to improve on to collaborate with others better.
